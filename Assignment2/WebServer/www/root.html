You can make the following GET requests

  • / — shows this page (root.html)
  • /random — returns index.html
  • /json — returns JSON for the /random request
  • /file/sample.html — returns the content of the file (simplified in this server)
  • /multiply?num1=3&num2=4 — multiplies two integers
  • /github?query=users/amehlhase316/repos — shows repo full_name, id, and owner.login
  • /password?length=16&symbols=true — generates a random password (length 8–64, symbols true/false)
  • /convert?value=72&from=F&to=C — converts temperature between C, F, and K
